**24시간 자동으로 돈을 버는 시스템** - AI 자동화 기반 수익화 전문 블로그

다크모드 테크 스타일의 미래지향적 홈페이지형 블로그입니다. AI 자동화, 패시브 인컴, 수익화 전략을 다루는 전문 콘텐츠 플랫폼입니다.

---

## ✨ 주요 특징

### 🎨 디자인
- ✅ **다크모드 & 사이버펑크 스타일** - 네온 블루/그린/퍼플 강조색
- ✅ **완전 반응형** - 모바일, 태블릿, 데스크톱 최적화
- ✅ **애니메이션 효과** - 스크롤 애니메이션, 카운터 애니메이션, 3D 호버 효과
- ✅ **네온 효과** - 실시간 네온 라인 생성, 그리드 배경 애니메이션

### 🚀 기능
- ✅ **히어로 섹션** - 임팩트 있는 랜딩 페이지
- ✅ **수익화 방법 카드** - 4가지 수익 모델 소개
- ✅ **블로그 포스트 그리드** - 6개 샘플 포스트 포함
- ✅ **통계 섹션** - 애니메이션 카운터로 실적 시각화
- ✅ **뉴스레터 구독** - 리드 생성 폼
- ✅ **스크롤 인터랙션** - Intersection Observer 활용

### 🛠️ 기술 스택
- **HTML5** - 시맨틱 마크업
- **CSS3** - Flexbox, Grid, Custom Properties
- **Vanilla JavaScript** - 프레임워크 없이 순수 JS
- **Google Fonts** - Orbitron (디스플레이), Noto Sans KR (본문)
- **Font Awesome 6** - 아이콘
